# Plan 5 - Phase 2 Feature Set (Months 7–18)

> Goal: Transform Malkiat from a portal into a real estate ecosystem. Activate B2B revenue through agent SaaS, open new verticals (short-term rental, developer projects, home finance), and apply AI to differentiate search, valuation, and lead quality.

---

## Strategic North Star for Phase 2

**Malkiat Phase 2 = "The operating system for Pakistan real estate."**

Phase 1 wins buyers and renters. Phase 2 wins agents, agencies, developers, and financial institutions as paying B2B customers — creating compounding revenue that doesn't depend on advertising alone.

---

## Phase 2 Feature Modules

### Module 7: Agent CRM & Agency SaaS (B2B Revenue Engine)

**Why this matters:** Every agent in Pakistan currently uses Zameen for discovery + a separate tool (Aiksol360, Broker92, WhatsApp) for CRM. Collapsing both into Malkiat creates stickiness and recurring B2B SaaS revenue.

#### 7.1 — Agent Lead Dashboard (CRM)

- All leads in one inbox: email, SMS, WhatsApp, in-platform inquiry
- Lead status pipeline: New → Contacted → Showing → Negotiating → Closed / Dead
- Lead notes + follow-up reminders
- Lead source attribution: which listing, which search query, which channel
- Lead export (CSV/Excel)
- Response time tracker (measured and shown on agent public profile)

#### 7.2 — SmartLeads™ (Behavior-Based Lead Intelligence)

- Analyze lead's browsing behavior: which areas viewed, price range engaged with, property types clicked
- Auto-tag lead preferences from behavior
- "Best match" suggestions: "This lead browsed DHA Phase 5 listings, 3-bed, PKR 3–5Cr range"
- Lead score (hot/warm/cold) based on activity recency and engagement depth

#### 7.3 — WhatsApp Business Integration

- Agent connects their WhatsApp Business account
- Inquiry notifications sent to agent's WhatsApp
- Templated quick-reply messages (follow-up, availability confirmation, showing request)
- Opt-in automation: "Reply 1 to confirm availability, 2 to mark as rented"
- Conversation history synced to CRM

#### 7.4 — Listing Boost Marketplace (Agent Facing)

- Credits system (buy credits, spend on boosts):
  - Featured Listing: Top of search for 7/14/30 days in selected area
  - Hot Listing: Badge + priority in discovery feed
  - Refresh Credit: Reset freshness timer without form re-submission
  - Story Credit: Short-form video slot on homepage / app feed
- Transparent labeling to users: "Sponsored" / "Featured" tag
- Spend analytics: cost per view, cost per inquiry for each boost

#### 7.5 — Agency Management Portal

- Agency admin role: manage all agent profiles, listings, and leads under one org
- Agent quota management: how many active listings per agent
- Team performance dashboard: leads by agent, listings by agent, response rate by agent
- Commission tracking (internal, not exposed to clients):
  - Deal tracking: property → client → deal stage → commission amount
  - Commission split rules between agents and agency
- Billing: agency-level subscription + credit top-ups

#### 7.6 — Listing Quality Coaching

- Quality score with actionable improvement tips (same as Bayut's Profolio™ model)
- "Your listing is missing: 3+ photos, floor plan, geo coordinates — add them to increase views by 3x"
- Historical performance graph per listing (views, inquiries over time)
- Comparison: "Your similar listings average 15 views; this one has 3 — here's why"

### Module 8: Developer Projects Microsite

**Why this matters:** Developer paid placements are a major revenue stream for Zameen. Each new project launch is a high-value B2B deal.

#### 8.1 — Developer Profile & Project Pages

- Branded developer page: logo, portfolio, past projects, track record
- Per-project microsite:
  - Location on map with master plan overlay
  - Payment plan table (installment breakdown)
  - Available unit types (1-bed, 2-bed, plot sizes, commercial)
  - Progress updates (% construction complete, photos by date)
  - Approval badges (CDA/LDA/RDA approval status, NOC links)
  - Brochure and floor plan downloads (gated behind contact form)

#### 8.2 — Developer Lead Capture

- Dedicated inquiry form per project
- Leads routed to developer's sales team CRM
- Lead analytics per project campaign

#### 8.3 — Investment Project Verification Badge

- Malkiat-verified badge for projects with confirmed legal approvals
- Checklist-based verification: NOC, developer registration, approved layout plan
- Visible on all listings under a verified project

### Module 9: AI-Powered Features

**Why this matters:** Redfin and Zillow's conversational search increased tour conversion by 35–47%. AI valuation achieves sub-3% error rates. These are now competitive necessities, not luxury features.

#### 9.1 — Conversational / Natural Language Search

- Input: "I want a 3 marla house in Bahria Town Phase 8 under 1.2 crore, near a park"
- NLP pipeline extracts: area, property type, price, area size, feature preference
- Follow-up refinement: "Do you want corner plots too?" → user refines
- Back-and-forth dialogue to narrow results
- Multilingual: English + Urdu + Romanized Urdu
- Accessible via web search bar, mobile app, and WhatsApp bot

#### 9.2 — AI Property Valuation (Malkiat Estimate)

- Inputs: property type, area (marla/sqft), beds/baths, location (city/zone/society), condition, amenities
- Model trained on historical Malkiat + scraped listing price data
- Output: estimated market value range + confidence band
- Used by sellers when pricing ("Price your listing")
- Used by buyers to see if asking price is fair ("Is this priced right?")
- Refreshed weekly as market data accumulates

#### 9.3 — AI Listing Description Generator

- Agent inputs basic details (beds, area, location, features)
- AI generates a professional listing description in English and/or Urdu
- Agent edits and approves before publish
- Reduces listing creation friction → increases listing quality

#### 9.4 — AI-Powered Lead Qualification

- Analyze incoming lead behavior before routing to agent
- Score lead: likelihood to transact, estimated budget range, seriousness signal
- Prioritize high-score leads in agent CRM inbox
- "Hot lead" badge for leads showing high engagement (multiple visits, saved, inquiry)

#### 9.5 — Smart Recommendations

- For buyers: "Homes you might love" based on search history and saved listings
- For agents: "Users browsing similar to your listings" (reach-out list suggestion)
- Email digest: "New listings matching your behavior this week"

### Module 10: Home Finance Integration

**Why this matters:** Graana is going in this direction, and it's a major search intent category in Pakistan (especially post-government housing schemes). Being the finance guide captures high-intent users.

#### 10.1 — Mortgage Calculator (Enhanced)

- Live inputs: property price, down payment %, tenure (years), bank
- Supports: conventional and Islamic (diminishing musharakah) repayment modes
- Output: monthly installment, total payment, total profit/interest, amortization schedule
- Partner banks: HBL, Meezan, Bank Alfalah, UBL, NBP, Bank of Punjab
- "Apply Now" CTA routes to bank's application or partner affiliate

#### 10.2 — Home Finance Guides (Content Hub)

- Curated guide library:
  - "How to get a home loan in Pakistan"
  - "Meezan Bank vs HBL for Islamic mortgages"
  - "Government housing schemes comparison (Apna Ghar, Roshan Apna Ghar, PM Naya Pakistan)"
  - "Step-by-step guide for NRPs buying property remotely"
- SEO value: high-intent search traffic acquisition
- Bank partner content (sponsored guides = revenue)

#### 10.3 — Finance Eligibility Estimator

- User inputs: monthly income, existing liabilities, employment type
- Output: estimated loan eligibility range across partner banks
- "Based on your income, you could qualify for up to PKR X crore"
- Soft lead capture: name + phone for bank partner referral

### Module 11: Short-Term Rental Vertical (Airbnb for Pakistan)

**Why this matters:** A blue ocean in Pakistan. No dominant local platform exists for guest houses, serviced apartments, and short-stay rentals in major cities.

#### 11.1 — STR Listing Type

- New listing intent: "Short-term rental" (distinct from long-term rent/sale)
- STR-specific attributes: available from/to dates, nightly rate, minimum stay, cancellation policy
- Amenities checklist: WiFi, AC, generator backup, kitchen, parking, security

#### 11.2 — Booking Engine

- Calendar availability display
- Instant book vs request-to-book toggle
- Guest messaging before booking
- Booking confirmation + auto-confirmation email
- Booking management: upcoming / past / cancelled

#### 11.3 — Payment Integration

- Online payment for STR bookings:
  - JazzCash / EasyPaisa (Pakistan mobile money — essential)
  - Debit/credit card via Stripe or HBL's payment gateway
  - Bank transfer option for longer stays
- Host payout automation (minus platform commission)
- Escrow-style hold: payment released 24h after check-in

#### 11.4 — Host Dashboard

- Calendar view: bookings, blocked dates, pricing by date
- Dynamic pricing suggestions (weekends, holidays)
- Guest reviews and ratings
- Earnings report: monthly, by property
- Multi-property management

#### 11.5 — Guest Trust Features

- Host identity verification (CNIC)
- Property verification visit (optional, for Malkiat Verified STR badge)
- Guest rating after each stay
- Security deposit pre-authorization

### Module 12: NRP / Overseas Pakistani Workflow

**Why this matters:** NRPs are a high-value, underserved segment. They want to invest or buy property from abroad but face trust, process, and payment friction that no current portal addresses.

#### 12.1 — NRP-Specific Search & Discovery

- "Investing from abroad" filter/mode
- Shows only verified listings and projects
- Legal approval status prominently displayed
- Agent profile: NRP experience badge (agents who've handled NRP transactions)

#### 12.2 — Remote Property Verification Service

- NRP requests a Malkiat-verified site visit for a listed property
- Malkiat sends a partner inspector to the property
- Inspector submits: photos, video walkthrough, legal document checks, neighborhood report
- Report delivered to NRP within 48–72h
- Pricing: PKR 5,000–15,000 per inspection (revenue stream)

#### 12.3 — Cross-Border Payment Support

- Integration with Roshan Digital Account / Roshan Apna Ghar scheme
- Bank transfer from international accounts (SWIFT reference guide)
- Token payment through escrow (SafeDealPro API integration)
- Legal document e-signing (DocuSign or local equivalent)

#### 12.4 — NRP Legal Resource Center

- Document guides: fard, registry, mutation, power of attorney from abroad
- NADRA CNIC/NICOP requirements
- Trusted lawyer referral network (vetted by Malkiat)

### Module 13: Neighborhood Intelligence (Full)

**Why this matters:** No Pakistan portal has this. First-mover advantage is clear. Buyers make major decisions based on neighborhood quality — which is currently invisible in listings.

#### 13.1 — Neighborhood Score

Each society/zone gets a computed score across:
- Schools: presence and rating of nearby schools (private/public/university)
- Healthcare: hospitals, clinics
- Worship: mosques, churches
- Shopping: markets, malls, commercial areas
- Transport: bus routes, proximity to motorway/main road
- Utilities: electricity load-shedding history (LESCO/IESCO zone data), gas availability
- Safety: user-reported rating, police station proximity

Displayed as a radar chart on society pages and listing detail.

#### 13.2 — Commute Calculator

- Input: listing location + user's workplace/school
- Output: estimated drive time, public transport time, distance
- Integrates Google Maps Distance Matrix API or OpenRouteService (open-source)

#### 13.3 — Price vs Neighborhood Analysis

- "Is this priced right for this area?" comparison
- Show median price per marla in this zone vs similar zones
- Price-per-sqft map overlay for premium users
